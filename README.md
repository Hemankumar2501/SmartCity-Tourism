# 🌍 SmartCity Tourism App
An AI-powered full-stack web application that generates personalized travel itineraries and smart city tourism guides instantly using Google's Gemini AI.

## 🚀 Live Demo
* **Frontend (Vercel):** https://smart-city-tourism.vercel.app/
* **Backend API (Render):** https://smartcity-tourism.onrender.com/docs

## ✨ Features
* **AI Itinerary Generation:** Get day-by-day travel plans customized to your destination using Gemini 2.5 Flash.
* **Budget & Expense Estimator:** Automatically calculates estimated costs for flights, hotels, food, and transport.
* **Modern UI/UX:** Fast, responsive, and beautiful frontend built with Next.js & Tailwind CSS.

## 🛠️ Tech Stack
Next.js, FastAPI, Google Generative AI (Gemini 2.5 Flash), Vercel, Render.
